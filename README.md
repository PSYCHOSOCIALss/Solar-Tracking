1. การทดสอบการเชื่อมต่อ Wi-Fi และ Blynk 
เชื่อมต่อ ESP8266 กับเครือข่าย Wi-Fi โดยตรวจสอบการเชื่อมต่อใน Serial Monitor หาก
เชื่อมต่อสำเร็จจะต้องแสดงข้อความ "Connected with IP: [IP address]" และสามารถ
เชื่อมต่อกับแอป Blynk ได้. 
ทดสอบฟังก์ชัน Blynk.run() และ timer.run() ใน loop() เพื ่อให้มั ่นใจว่าข้อมูลจากเซ็นเซอร์
ถูกส่งไปยังแอป Blynk ทุกๆ 10 วินาที. 
2. การทดสอบเซ็นเซอร์แสง (BH1750) 
ตรวจสอบว่าเซ็นเซอร์แสงทำงานโดยการอ่านค่าความเข้มแสงจากเซ็นเซอร์ทุกตัว (lux[]) 
และแสดงผลบน Serial Monitor และ LCD. 
ตรวจสอบความถูกต้องของข้อมูลที่แสดงผล เช่น ค่าความเข้มแสงในหน่วย "lux". 
3. การทดสอบการควบคุมเซอร์โว (Servo) 
ทดสอบการควบคุมเซอร์โวโดยการปรับค่ามุมผ่าน Blynk (การควบคุมมุมโดยใช้จอยสติ ๊กใน 
Virtual Pin 4 และ 5). 
ตรวจสอบว่ามุมของเซอร์โวเปลี่ยนแปลงตามค่าที ่ส่งจากแอป Blynk และตรวจสอบการหมุน
ของเซอร์โวทั ้งสองแกน (X และ Y). 
4. การทดสอบการเลือกช่อง TCA9548A I2C Multiplexer 
ตรวจสอบการเลือกช่องสำหรับเซ็นเซอร์แสงต่างๆ โดยการใช้ฟังก์ชัน tcaSelect() ซึ่งจะสลับ
การเชื่อมต่อเซ็นเซอร์แสงแต่ละตัว. 
ให้แน่ใจว่าเซ็นเซอร์แสงแต่ละตัวสามารถอ่านค่าความเข้มแสงได้อย่างถูกต้องเมื่อเลือกช่องที่ข้อง. 
5. การทดสอบการแสดงผลบน LCD 
ตรวจสอบว่าค่าความเข้มแสงจากเซ็นเซอร์ทั้งหมด (lux[0] ถึง lux[3]) แสดงผลอย่างถูกต้อง
บนจอ LCD และอัปเดตทุกๆ ครั้งที ่ค่าแสงเปลี่ยนแปลง. 
6. การทดสอบฟังก์ชันการปรับมุมเซอร์โว (Auto Mode) 
ในโหมดอัตโนมัติ (หาก manualControl == false), ให้ตรวจสอบว่าเซอร์โวปรับมุมตามค่า
ความแตกต่างของแสงในแต่ละแกน (X และ Y) และส่งค่ามุมไปยัง Blynk ทุกๆ 10 วินาที. 
7. การทดสอบฟังก์ชันการควบคุมด้วยมือ (Manual Mode) 
เมื่อผู ้ใช้ควบคุมมุมเซอร์โวด้วยจอยสติ๊กในแอป Blynk, ตรวจสอบว่าเซอร์โวเปลี่ยนมุมตาม
ค่าที่ได้รับจากการควบคุมและไม่เปลี่ยนแปลงในโหมดอัตโนมัติ. 
8. การทดสอบการส่งข้อมูลไปยัง Blynk 
ตรวจสอบว่าข้อมูลจากเซ็นเซอร์แสงและมุมเซอร์โวถูกส่งไปยังแอป Blynk อย่างถูกต้อง และ
สามารถแสดงผลได้ทันทีเมื่อมีการอัปเดต.
